# toast组件
----
#### 基础用法
``` js
     this.$toast.text(content, direction)
```
#### 适配：移动端、PC端
#### 属性
| 字段名称 | 字段说明 | 类型 | 必填 | 备注 |
| ------ | ------ | ------ | ------ | ------ |
| content | 内容 | string | y |
| direction | 位置 | string | n |可选：top、center、bottom, 默认center|
